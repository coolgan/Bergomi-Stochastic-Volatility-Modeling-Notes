# Bergomi 随机波动率建模中文精读笔记

本仓库托管 Lorenzo Bergomi《Stochastic Volatility Modeling》的非官方中文精读笔记静态网页。内容覆盖局部波动率、随机波动率导论、方差互换、Heston 模型、前向方差模型、随机波动率模型的微笑、股权微笑成因、多资产随机波动率以及局部随机波动率模型等主题。

## 项目简介

本项目面向希望系统学习波动率建模、期权定价与交易台风险管理框架的读者。网页由 Markdown 笔记自动构建，数学公式在构建时通过 KaTeX 渲染，图片和字体均随站点打包，不依赖外部 CDN。

## 部署

仓库根目录即静态站点根目录。Cloudflare Pages 可直接连接本仓库，构建命令留空，输出目录使用根目录。
